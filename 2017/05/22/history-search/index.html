简单的实现历史搜索功能

作者:super J726    发布于: 2017-05-22

本文主要介绍如何简单的实现历史搜索功能。

思路及实现过程：
1.获取本地储存的数据，并解析成一个数组，这里用到H5的一个方法：localStorage。

localStorage与sessionStorage区别在于前者可以永久保存，后者则是在浏览器关闭之后，数据就会被清空。

其中 searchHistory 为自定义的本地数据数组名称。

将获取数据函数封装，在提交搜索函数内调用。

 getSearchHistory() {     //读取本地储存的数据

                var arr = localStorage.searchHistory ? JSON.parse(localStorage.searchHistory) : [ ];

                return arr;

        }
2.在执行onSubmit命令提交搜索时进行判断，判断用户输入的关键词是否与本地储存的数据中其中一项重复，若判断不重复则存入本地数据中，重复则不保存。

 onSubmit() {  //执行提交搜索函数
            var searchName = this.keywords;//定义变量，将输入框内的关键词赋值给变量
            console.log('this.searchName', searchName)
            if (searchName == "") {//判断关键词是否为空
                this.showNowords = true;
                return;
            } else {
                this.showNowords = false;
                var data = this.getSearchHistory();//定义变量，将本地储存的数据赋值给变量
                if (data.length == 0) {//判断本地储存是否无数据
                        data.push(searchName);//判断成功，将关键词变量的值存入本地数据数组中
                } else {
                        var flag = true;//定义一个标识，用于决定是否将输入框内的值存入本地
                        for (var i = 0; i < data.length; i++) {//循环本地储存的数据，逐一与输入框内的值对比
                                if (data[i] == searchName) {//判断输入框内的值是否与本地储存的值有相同
                                        flag = false;
                                }
                        }
                        if (flag) {//判断如果标识为true执行下一步
                                data.push(searchName)//判断成功，将关键词变量的值存入本地数据数组中
                            }
                 }
                this.saveSearchHistory(data);//执行保存本地储存数据函数
            }
     }
3.执行保存本地储存数据的函数。

将保存数据函数封装，在提交搜索函数内调用。


 saveSearchHistory(data) {  //保存本地储存的数据

                    localStorage.searchHistory = JSON.stringify(data);

            }
4.将数据在页面中展示出来。

  <span v-for="value in data" @click="goSearchHistory($event)">{{value}}</span>

data就是储存在本地的数据数组，循环value，在页面中展示

5.清除历史搜索记录。

清空本地储存数据，直接用localStorage里的方法：localStorage.clear();

并重新获取一下本地的数据以使页面数据得到更新。

简单的历史搜索就做好啦！
